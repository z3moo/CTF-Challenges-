# American Bee

**Category:** Forensics / USB Traffic Analysis  
**Difficulty:** Easy

## Scenario

> "I swear I was just typing a report!"

A suspect was apprehended, but their computer was locked. However, we managed to capture the USB traffic from a device they were using just before the arrest. We believe they were typing a secret message using a specific input device.

- **Solution:** Located in the `solution` folder. Note that the solution is encrypted with the flag itself.
